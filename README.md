# LiAuto_Network_Scanner

Программа предназначена для проверки доступности серверов обновления(OTA) и управления(APP) автомобилем Lixiang.

# Перед началом использования изучите файл **[User_Manual](User_Manual.md)** 

*Пример:* когда не возможно получить мастер-акаунт (МА) и машина не управляется с мобильного приложения, а в инженерном меню (box715) такие данные: 

<img src="Engineering_menu.png" alt="Инженерное меню" width="600" height="400"> 

В этом случае вывод программы будет такой, если не хуже: 

<img src="Доступ блокирован.png" alt="Инженерное меню" width="900" height="600"> 

Когда сервера доступны, машина может обновляться (кроме неиспраности самого авто), а приложение мобильного телефона может управлять машиной и отображать ее состояние, вывод программы будет такой:

<img src="Сервера доступны.png" alt="Инженерное меню" width="900" height="600">
